## 在 CodePen 上跑

1. 開新 Pen → ⚙️ HTML Settings
2. 把 `widgets/tarot/index.html` 全部複製貼到 Pen 的 HTML 區
3. CSS / JS 區留空（已內嵌在 HTML 裡）
4. 儲存即可

## 本機預覽

直接用瀏覽器打開 `widgets/tarot/index.html` 即可。
